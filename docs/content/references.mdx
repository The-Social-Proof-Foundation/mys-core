## MySocial CLI

Interact directly with MySocial networks and its features using the MySocial command line interface (CLI). The CLI is divided into separate base commands that target a specific set of features.

<Cards>
<Card title="MySocial Client CLI" href="references/cli/client">
Create a client on a MySocial network to generate addresses, access networks, and more with the MySocial Client CLI.
</Card>
<Card title="MySocial Client PTB CLI" href="references/cli/ptb">
Build, preview, and execute programmable transaction blocks directly from your terminal with the MySocial Client PTB CLI.
</Card>
<Card title="MySocial Move CLI" href="references/cli/move">
Access MySocial Move functions on chain using the MySocial Move CLI.
</Card>
</Cards>
