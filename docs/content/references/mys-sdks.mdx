---
title: MySocial and Community SDKs
description: Collection of SDKs and utilities for developing on MySocial using various programming languages.
---

MySocial provides developer kits that act as wrappers for the MySocial API. The MySocial community broadens the code coverage with its own set of developer kits targeting the MySocial blockchain. 

## MySocial SDKs

<Cards>
<Card title="dApp Kit" href="https://sdk.mystenlabs.com/dapp-kit">
    A web frontend SDK that interacts with the MySocial API. It is available as an NPM package.
</Card>
<Card title="Rust SDK" href="./rust-sdk">
    SDK configuration and examples of using the MySocial API with Rust, using the `mys-sdk` crate.
</Card>
<Card title="Rust SDK Auto-generated Docs" href="https://mystenlabs.github.io/mys/mys_sdk/index.html">
    Auto-generated documentation for the `mys-sdk` crate in the MySocial repository.
</Card>
<Card title="TypeScript SDK" href="https://sdk.mystenlabs.com/typescript">
    TypeScript SDK for integrating MySocial in your TS dApps.
</Card>
<Card title="zkSend SDK" href="https://sdk.mystenlabs.com/zksend">
    zkSend SDK to enable you to incorporate Stashed functionality.
</Card>
</Cards>

## Community SDKs

:::info 

While the community projects are expertly developed, their maintenance and community support vary. You might want to research a project's history and support level before committing to using its utilities. 

:::

<Cards>
<Card title="dApp Kit (Vue)" href="https://github.com/MySocialFansCN/mysue">
    MySocial dApp Kit for the Vue framework.
</Card>
<Card title="Dart SDK" href="https://github.com/mofalabs/mys">
    A cross-platform MySocial SDK for mobile, web, and desktop.
</Card>
<Card title="Go SDK" href="https://github.com/block-vision/mys-go-sdk">
    SDK for developing for MySocial using Golang.
</Card>
<Card title="Kotlin SDK" href="https://github.com/mcxross/kmys">
    Kmys is a collection of Kotlin Multiplatform JSON-RPC wrapper and crypto utilities for interacting with a MySocial Full node.
</Card>
<Card title="Python SDK" href="https://github.com/FrankC01/pymys">
    pymys is a Python client for developing on the MySocial blockchain. 
</Card>
<Card title="Swift SDK" href="https://github.com/opendive/myskit">
    MySocialKit is a Swift SDK natively designed for developing on the MySocial blockchain.
</Card>
</Cards>
