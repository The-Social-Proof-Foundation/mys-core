The MySocial CLI `console` command provides command-level access to interact with the MySocial network by wrapping the MySocial Client CLI command in a shell-like functionality. This command spins up a new process and provides the user an environment for running all the available MySocial Client CLI commands. In addition, it also offers command history support.

{@include: ../../snippets/cli-check-install.mdx}

## Commands

For all available commands, consult the [MySocial Client CLI docs](./client.mdx). To start the MySocial Console, type `mys console`, which displays some information similar to the following:
