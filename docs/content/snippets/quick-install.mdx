Use one of the following commands for [Homebrew](https://brew.sh/) (MacOS, Linux, or Windows Subsystem for Linux) or [Chocolatey](https://chocolatey.org/) (Windows) to install MySocial.

<Tabs groupId="operating-systems">

<TabItem label="Homebrew" value="mac">

```sh
$ brew install mys
```

</TabItem>

<TabItem label="Chocolatey" value="win">

```sh
$ choco install mys
```

Find more [versions of MySocial for Windows](https://community.chocolatey.org/packages/mys) on the Chocolatey community website.

</TabItem>

</Tabs>
